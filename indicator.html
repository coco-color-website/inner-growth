主动SKDJ主力散户买卖点

同花顺副图指标

结合主动买卖、SKDJ抄底逃顶、主力散户走势三个维度，生成综合买入卖出信号。

📌 指标逻辑

  • SKDJ 抄底：K 线上穿 D 线，且 K 值低于 20，视为超卖区反弹信号。
  • SKDJ 逃顶：D 线上穿 K 线，且 K 值高于 80，视为超买区回落信号。
  • 主动买卖：主动买入量大于主动卖出量，说明资金主动进攻。
  • 主力散户：主力线上穿散户线，说明大资金进场；下穿则离场。
  • 买入信号：SKDJ 抄底 + 主动买占优 + 主力进场，三者共振。
  • 卖出信号：SKDJ 逃顶 + 主动卖占优 + 主力离场，三者共振。

// 指标名称：主动_SKDJ_主力散户综合买卖点
// 适用软件：同花顺
// 指标类型：副图指标
// 核心逻辑：
//   1. SKDJ 低位金叉 -> 抄底信号
//   2. SKDJ 高位死叉 -> 逃顶信号
//   3. 主动买 > 主动卖 -> 资金主动进攻
//   4. 主力线 > 散户线且金叉 -> 主力进场
//   5. 买入：SKDJ抄底 + 主动买占优 + 主力进场
//   6. 卖出：SKDJ逃顶 + 主动卖占优 + 主力离场

// 参数设置
N:=9;
M:=3;
SKDJ_LOW:=20;
SKDJ_HIGH:=80;

// SKDJ 指标
LOWV:=LLV(LOW,N);
HIGHV:=HHV(HIGH,N);
RSV:=(CLOSE-LOWV)/(HIGHV-LOWV+0.000001)*100;
K:SMA(RSV,M,1),COLORWHITE,LINETHICK2;
D:SMA(K,M,1),COLORYELLOW,LINETHICK2;
J:3*K-2*D,COLORMAGENTA,LINETHICK1;

// 画超买超卖区间参考线
SKDJ_HIGH_LINE:SKDJ_HIGH,COLORGREEN,DOTLINE;
SKDJ_LOW_LINE:SKDJ_LOW,COLORRED,DOTLINE;

// 主动买卖指标
主动买:IF(DYNAINFO(23)>0, DYNAINFO(23), BUYVOL);
主动卖:IF(DYNAINFO(24)>0, DYNAINFO(24), SELLVOL);
主动差:主动买-主动卖,COLORCYAN,LINETHICK1;
主动差MA:EMA(主动差,5),COLORGREEN,LINETHICK1;

// 主力散户走势指标
主力净:IF(LARGEBUYVOL>0 AND LARGESELLVOL>0, LARGEBUYVOL-LARGESELLVOL,
           IF(C>O, VOL*(C-O)/(H-L+0.000001), -VOL*(O-C)/(H-L+0.000001)));

散户净:IF(SMALLBUYVOL>0 AND SMALLSELLVOL>0, SMALLBUYVOL-SMALLSELLVOL,
           IF(C>O, -VOL*(C-O)/(H-L+0.000001)*0.3, VOL*(O-C)/(H-L+0.000001)*0.3));

主力线:EMA(主力净,5),COLORRED,LINETHICK2;
散户线:EMA(散户净,5),COLORBLUE,LINETHICK1;

// 买卖信号
抄底:=CROSS(K,D) AND K<SKDJ_LOW;
逃顶:=CROSS(D,K) AND K>SKDJ_HIGH;
主力进场:=CROSS(主力线,散户线);
主力离场:=CROSS(散户线,主力线);
主动买占优:=主动差>主动差MA;
主动卖占优:=主动差<主动差MA;

买入信号:抄底 AND 主动买占优 AND 主力进场,NODRAW;
卖出信号:逃顶 AND 主动卖占优 AND 主力离场,NODRAW;

DRAWICON(买入信号, K, 1);
DRAWICON(卖出信号, K, 2);
DRAWTEXT(买入信号, K-5, '买'), COLORRED;
DRAWTEXT(卖出信号, K+5, '卖'), COLORGREEN;

⚙️ 安装步骤

  1. 打开同花顺客户端，点击顶部菜单【工具】→【公式管理】。
  2. 在公式管理器中，选择【技术指标】→【新建】。
  3. 指标名称填写：主动SKDJ主力散户买卖点。
  4. 指标类型选择：副图指标。
  5. 把上面的源码全部复制进去，点击【测试公式】→【确定】。
  6. 打开任意股票 K 线图，在副图指标中选择刚保存的指标即可。

⚠️ 注意事项

  • 本指标效果最佳需 Level-2 行情支持主动买卖/大单数据。
  • 若无 Level-2，指标会自动 fallback 到外盘内盘和价格估算方案，精度下降。
  • 任何技术指标都有滞后性和误判，建议结合 K 线形态、成交量、大盘环境综合判断。
  • 不要根据单一信号满仓操作，建议作为辅助参考工具。
